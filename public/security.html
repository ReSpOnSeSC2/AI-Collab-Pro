‚ùì Frequently Asked Questions

‚ñ∂ Why should I trust you with my API keys?
We use industry-standard encryption (AES-256-GCM) to protect your keys. They're encrypted before leaving your browser, and we never have access to the decrypted versions. Additionally, our code is open source, so you can verify our security practices yourself.
‚ñ∂ Can I use the service without providing my API keys?
Yes! You have three options: (1) Use our local storage mode where keys never leave your browser, (2) Use our test mode to try the service without saving keys, or (3) Use system-provided keys with usage limits.
‚ñ∂ What happens if there's a data breach?
Even in the unlikely event of a breach, your API keys remain safe because they're encrypted with keys that only you control. An attacker would need both the encrypted data AND your encryption key, which we don't store together.
‚ñ∂ How do I delete my API keys?
You can delete your API keys at any time from the API Keys management page. Deletion is immediate and permanent. We also provide an option to export your data before deletion.
‚ñ∂ Do you sell or share API keys with third parties?
Absolutely not. We never sell, share, or use your API keys for any purpose other than providing you with our AI collaboration service. Our business model is based on subscriptions, not data sales.
